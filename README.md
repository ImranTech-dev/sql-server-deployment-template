# SQL Server Deployment Template

A production-ready SQL Server deployment structure for organizing database changes using version folders.

## Features

- Version-based deployment
- Organized SQL scripts
- Rollback support
- Easy to maintain
- Production-friendly folder structure

## Folder Structure

```
DatabaseDeployment
│
├── V1.0.0
├── V1.0.1
├── Rollback
└── README.md
```

## Technologies

- SQL Server
- T-SQL
